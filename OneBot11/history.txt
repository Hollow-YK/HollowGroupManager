多群联动管理Bot — 版本历史
============================

V1.0.5 (2026-06-25)
--------------------
▸ 多配置架构：单管理组 → 多配置独立存储（data/<配置名>/punish/）
▸ 命令配置系统：data/command.json 全局 + 各配置 command.json 覆盖，支持别名
▸ 权限模型扩展：三级（0/1/-1）→ 多级（0/≥1/-1），各命令支持 min_level
▸ 指令变更：/group → /config（new/rename/notify/set/remove/group 子命令）
▸ help 重构：卡片化分组（圆角背景），添加 AGPL v3 声明
▸ 新增旧版数据迁移工具 tools/migrate.py
▸ 代码分层：将指令处理拆分为 render.py / commands.py

V1.0.4 (2026-06-24)
--------------------
▸ 初始发布
▸ 基于 OneBot v11 协议的全新 Python 实现，支持在 Linux / Windows / macOS 任意平台运行
▸ 通过反向 WebSocket 与 OneBot 实现端通信，异步处理事件与指令
▸ 支持踢人、禁言、警告、撤销、记录查询等完整处罚操作
▸ 支持黑名单自动拦截入群
▸ 与 QFun Java 版指令集、数据格式、权限模型完全兼容，数据文件可互换
▸ 基于唤醒词前缀的指令系统，支持 @某人 方式指定目标
▸ 记录查询支持汇总统计与图片表格详情（含彩色状态标注）
▸ 处罚操作前检查成员在群状态，避免无效操作
▸ 撤销禁言时自动解禁，撤销踢黑时自动移出黑名单
