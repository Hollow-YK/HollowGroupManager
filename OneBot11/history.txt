多群联动管理Bot — 版本历史
============================

V1.0.4 (2026-06-24)
--------------------
▸ 初始发布
▸ 基于 OneBot v11 协议的全新 Python 实现，支持在 Linux / Windows / macOS 任意平台运行
▸ 通过反向 WebSocket 与 OneBot 实现端通信，异步处理事件与指令
▸ 支持踢人、禁言、警告、撤销、记录查询等完整处罚操作
▸ 支持黑名单自动拦截入群
▸ 与 QFun Java 版指令集、数据格式、权限模型完全兼容，数据文件可互换
▸ 基于唤醒词前缀的指令系统，支持 @某人 方式指定目标
▸ 记录查询支持汇总统计与图片表格详情（含彩色状态标注）
▸ 处罚操作前检查成员在群状态，避免无效操作
▸ 撤销禁言时自动解禁，撤销踢黑时自动移出黑名单
